UX Design – VR Game Experience

University Project · UX Design with Creative Technologies · VR / Web

Brief Persona Moodboard Sketches Storyboard UI Mockups Final Website

Project Brief

This project explores the design and promotion of a VR-based game experience aimed at increasing user engagement through immersive interaction. The pbriefroblem addressed was how traditional games often lack physical immersion and presence.

The proposed solution is a VR game that allows users to interact with a virtual environment in a more intuitive and embodied way. A responsive promotional website was designed to communicate the experience, highlight its features, and allow users to pre-order the game.

User Persona

A primary user persona was created to represent the target audience of casual and enthusiast gamers interested in VR technology and learn more. The persona helped guide design decisions by focusing on user motivations, goals, frustrations, and technical familiarity.

Jamie Taylor

“Learning is way more fun when I can actually see and do things.”

  • Age: 14
  • Location: Leeds, UK
  • Education: Year 9 Student
  • Personality: Curious, tech-savvy, easily distracted

Goals & Motivations

  • Understand the solar system visually
  • Learn through interaction and immersion
  • Build confidence using VR technology

Pain Points

  • Textbooks feel boring and overwhelming
  • 2D diagrams make space hard to visualise
  • Loses focus without interaction

User Scenario

Jamie discovers CosmoQuest VR through a YouTube advert and visits the website to learn more. After pre-ordering, Jamie uses the VR headset at school to explore planets interactively, complete challenges, and earn a digital badge — increasing confidence and interest in science.

Moodboard

The moodboard defined the visual direction of the VR game and promotional website. It focused on immersive environments, futuristic UI elements, and a dark, atmospheric colour palette inspired by existing VR games.

Sketches

Early sketches were produced to explore layout ideas for both the VR interface and the promotional website. These sketches allowed rapid iteration before moving into higher fidelity designs.

Storyboard

A storyboard was created to visualise the user journey, from discovering the promotional website to using the VR game. This helped ensure a clear start point, progression, and end goal for the experience.

UI Mockups

UI mockups were developed using Canva to demonstrate how users interact with the website.

The final outcome is a responsive promotional website developed using HTML and CSS. Screenshots are presented here to demonstrate layout, responsiveness, and visual consistency across devices.

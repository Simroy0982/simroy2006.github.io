Project Brief

This project required the development of a simple JavaScript-based library management system. The aim was to demonstrate core programming concepts such as conditional logic, functions, arrays, and basic user interaction.

The system allows users to view all books, check availability, search for specific titles, borrow books, and return them. Emphasis was placed on logical structure and clarity rather than visual complexity.

Use Case Diagram

A use case diagram was created to define user interactions with the library system and identify the key system functions.

System Flowcharts

Flowcharts were used to plan the logic behind each system function, ensuring clarity before implementation.

Final System Interface

The final system demonstrates all required functions, including displaying books, checking availability, searching titles, borrowing, and returning books.
